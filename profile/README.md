# OSCAT - Open Source Community for Automation Technology 👋
<img width="3333" height="304" alt="oscat_logoleiste" src="https://github.com/user-attachments/assets/f6f1fd89-2ca6-438a-9637-c38cd724393d" />


**[DE]** Willkommen bei **OSCAT**! Wir sind eine offene Gemeinschaft, die hochwertige Open-Source-Funktionsbibliotheken für die industrielle und gebäudetechnische Automation (IEC 61131-3) entwickelt.

**[EN]** Welcome to **OSCAT**! We are an open-source community developing high-quality function libraries for industrial and building automation (IEC 61131-3).

---

## 📚 Dokumentation / Documentation

**[DE]** Die gesamte Dokumentation ist unter **[oscat.readthedocs.io](https://oscat.readthedocs.io)** verfügbar.

**[EN]** All documentation is available at **[oscat.readthedocs.io](https://oscat.readthedocs.io)**.

### Unsere Bibliotheken / Our Libraries

| Projekt / Project | Beschreibung / Description |
| :--- | :--- |
| 📘 **[OSCAT Basic](https://oscat-basic.readthedocs.io)** | **[DE]** Mathematische Funktionen, Logik, String-Verarbeitung, Regelungstechnik und mehr.<br>**[EN]** Mathematical functions, logic, string processing, control engineering and more. |
| 🌐 **[OSCAT Network](https://oscat-network.readthedocs.io)** | **[DE]** Netzwerkprotokolle (TCP/IP, HTTP, FTP), Web Services und Wetterdaten.<br>**[EN]** Network protocols (TCP/IP, HTTP, FTP), web services and weather data. |
| 🏢 **[OSCAT Building](https://oscat-building.readthedocs.io)** | **[DE]** HLK, Jalousiesteuerung, Beleuchtung und Gebäudeautomation.<br>**[EN]** HVAC, blind control, lighting and building automation. |
